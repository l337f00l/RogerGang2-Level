Syncing ExAnimation

Despite how it may look in Lunar Magic, different sets of animated objects in the vanilla game are on different animation frames, so if you were looking to sync up ExAnimation with those objects (e.g. an ExAnimation sideways muncher with a vanilla muncher) it is worth bearing in mind those different timings.

Map 16

Point of Advice

You should use Lunar Magic's "Add Objects" window as much as possible to construct your levels instead of placing tiles directly from map 16. Objects that are composed of several tiles take up fewer bytes in-ROM when placed in levels.

When you use Direct Map 16 (i.e. placing tiles directly in your level from the 16x16 editor), each tile placed in a level counts as its own object in the level and this can dramatically increase the bytesize of levels if there are many many tiles.

Page 03

This page has the tiles for the screen-scrolling pipes and global line guides laid out in a fairly intuitive way. Includes optional ExGraphic screen-scrolling pipe corners.

Page 05

An optional page of extra vanilla tiles, some of which are graphics header dependent, that isn't inserted into the baserom. A map 16 file to import can be found in resources/map 16.

Pages 40,41

Tiles for all the in-level text seen in the Info Level. These pages can safely be deleted from your ROM if you delete that level from your project.

Page 50

Tiles for ExGraphic big bush and big pipe tiles. A map 16 file to import these tiles can be found in resources/map 16.
